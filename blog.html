Question-1: What are the different ways to select an element in the DOM?

getElementsByTagName(), getElementsByClassName(), getElementById(), querySelectorAll(), querySelector()

Question-2: What is the difference between innerHTML, innerText, and textContent?

- innerHTML returns all the content inside an element including html tags, texts, comments and everything.
- innerText returns only the text content inside an element, which are belongs to a html tag. It also consider the style of the text content.
- textContent ignores the style of the text and only returns the text content inside an element, which are belongs to a html tag.

Question-3: What is event delegation in the DOM?

Event delegation basically means adding a single event listener to parent element, instead of adding separate event listeners to each child element. It helps to achieve trigger events for child elements, even we need an event listener on the child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling basically means passing events from child elements to parent elements. For example, if we add event listeners for both child and parent element, and if an event occurs in child element, it will automatically trigger the event of parent element and the event of parent element will also take place.

Question-5: How do you create, add, and remove elements using JavaScript?

createElement() - create a HTML element
appendChild() - add a HTML element
removeChild() - remove a HTML element
